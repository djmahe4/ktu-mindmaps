# ⚡ Fast Conditions Cheat Sheet

## 🔹 1. Single Loop

### ✅ Linear → **O(n)**

👉 Condition:

* `i++`, `i--`, `i += k`, `i -= k`

💡 Rule:

> Moves one step each time

**Example**

```c
for(i=1; i<=n; i++)
```

---

### ✅ Logarithmic → **O(log n)**

👉 Condition:

* `i *= c`, `i /= c` (c > 1)

💡 Rule:

> Value grows/shrinks exponentially

**Example**

```c
for(i=1; i<=n; i*=2)
```

---

## 🔹 2. Nested Loops

### ✅ Independent → **O(n²)**

👉 Condition:

* Inner loop does **not depend on outer**

💡 Rule:

> Full loop runs every time

**Example**

```c
for(i=1; i<=n; i++)
  for(j=1; j<=n; j++)
```

---

### ✅ Dependent → **O(n²)** (still)

👉 Condition:

* Inner loop depends on `i` (like `j <= i`)

💡 Rule:

> Forms triangle → sum = n²/2

**Example**

```c
for(i=1; i<=n; i++)
  for(j=1; j<=i; j++)
```

---

## 🔹 3. Mixed Pattern

### ✅ Linear + Log → **O(n log n)**

👉 Condition:

* One loop linear, inner loop logarithmic

**Example**

```c
for(i=1; i<=n; i++)
  for(j=1; j<=n; j*=2)
```

---

## 🔹 4. Quick Recognition Rules

* ➕ Add loops → **O(n + n) = O(n)**
* ✖️ Nested loops → **multiply → O(n × n) = O(n²)**
* 🔁 Halving/doubling → **log n**
* 🔺 Triangle loops → **O(n²)**
